# 網站品牌風格指南

此文件保存 `yuchienpsy.com` 的長期品牌風格，供網站品質管理團隊、插畫家 AI、內容編輯 AI、SEO 策略 AI、網頁工程 AI 與品牌一致性 QA AI 使用。

## 品牌氣質

- 溫暖。
- 安定。
- 可信任。
- 有陪伴感。
- 清楚、有邊界。
- 不過度商業化。
- 不使用療效保證式語氣。

## 色彩方向

- 米白、暖棕、鼠尾草綠、低飽和粉橘為主要方向。
- 可使用低飽和藍灰、紫灰作為輔助色。
- 避免高飽和、強烈對比、醫療冷白或科技感過重的藍紫。

## 插圖方向

- 柔和手繪或編輯插畫感。
- 人物自然、多元、不刻板。
- 心理議題不恐怖化、不破碎化、不污名化。
- 原則上圖片內不放文字。
- 主題圖需要能辨識主題，但仍與整套網站像同一家人。

## 文案方向

- 用語清楚、溫暖、保守。
- 避免「一定改善」「立即療癒」「保證有效」等承諾。
- 清楚區分心理健康科普、個別諮商、講座合作與行政資訊。
- 高風險主題需保留求助、轉介或專業判斷邊界。

## SEO 方向

- SEO 不得凌駕專業形象與內容準確性。
- title 與 meta description 要符合搜尋意圖，但不可製造焦慮或承諾療效。
- 專欄 SEO 可聚焦心理健康教育與主題探索。
- 講座合作 SEO 可聚焦合作單位、講座主題、心理健康教育與行政窗口需求。

## 品質驗收原則

- 技術正確不等於品牌通過。
- 圖片裁切、主體焦點、人物表情、手機版呈現都要看最後效果。
- 文案、SEO、插圖與排版需共同形成同一品牌語氣。
- 任何重大視覺方向變更需使用者確認。
