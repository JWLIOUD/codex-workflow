## 回答方式

Q01、Q02 已完成。Q03–Q41 可直接在選項前的方框填入 `x`：

```text
- [x] A：採用建議
- [ ] B：保留原文
```

- 每題原則上只能勾選一個選項。
- 選擇「其他版本」時，請直接填寫文字。
- 完成後告訴需求分析 AI：「問卷填寫完成」。
