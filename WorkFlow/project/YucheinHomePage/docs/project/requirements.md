# 已確認需求

網頁工程 AI 與網頁測試 AI 應以本文件為正式需求依據，並先閱讀 `handoff.md`、`site-inventory.md` 與 `content-review.md`。

## 專案目標

- 完成使用者既有心理專欄文案的全部網頁。
- 讓每篇文章具備獨立網址、搜尋引擎可索引的內容與完整站內導覽。
- 增加非品牌搜尋字詞帶來的曝光與流量。
- 不由 AI 創作新的文章正文。

## 主要受眾

- 透過 Google 搜尋心理健康、職場壓力、成癮、界線、性騷擾、關係與自我照顧資訊的讀者。
- 閱讀文章後希望進一步了解心理師或諮商服務的潛在個案。

## 希望訪客採取的行動

- 從搜尋結果進入單篇文章。
- 繼續閱讀同系列或相關文章。
- 返回心理專欄首頁或了解心理師背景。
- 有需要時透過 LINE 詢問或預約諮商。

## 本次修改範圍

- 盤點並拆分 `專欄文章.txt` 中的全部文章。
- 參考 `專欄文章.docx` 的內容層級建立網頁排版。
- 建立文章資料清單、系列頁、其他文章分類及所有文章詳頁。
- 更新專欄首頁、內部連結、SEO metadata 與結構化資料。
- 修正目前的 404、檔案路徑、canonical 與缺少 CSS 問題。
- 建立疑似錯字審閱清單。
- 完成桌機、手機、連結、SEO 與可及性驗收。

## 不在本次範圍

- AI 創作新的文章正文。
- 未經使用者確認直接修正文案疑似錯字或改變原意。
- 講座合作頁的大型改版。
- 諮商服務內容與首頁其他區塊的大型重構。
- 沒有足夠內容的主題聚合頁。
- 虛構文章發表日期、案例、資料來源或專業聲明。
