# 手機／桌機截圖視覺 QA（2026-06-21）

## 執行方式

本輪由使用者先在 Chrome 開啟網站與手機裝置模式，再由總管 AI 使用 Windows 螢幕擷取保存截圖。

截圖位置：

- `quality-screenshots-2026-06-21/`

受驗頁面：

- 首頁
- 心理專欄列表
- 職場系列頁
- 成癮單篇文章
- 講座合作區塊

限制：

- 手機截圖包含 Chrome DevTools 右側區域；視覺判斷只採左側手機 viewport。
- 桌機截圖為 Chrome 實際桌機寬版畫面。

## 總體質感判斷

結論：目前網站沒有質感崩壞，整體已達到可公開作為心理師個人品牌網站的水準。

目前最強的視覺頁面是：

1. 首頁桌機版
2. 心理專欄桌機版
3. 系列頁桌機版

目前最需要補強的方向是：

1. 手機版部分頁面的首屏視覺出現較晚。
2. 單篇文章首屏缺少主題圖，品牌記憶點偏弱。
3. 講座合作目前像首頁段落，不像未來可獨立推廣的合作 landing page。

## 分頁驗收

### 首頁

桌機狀態：PASS。

優點：

- 真人照、暖色背景、主標題與 CTA 組合成熟。
- 第一眼有心理師個人品牌感，不像模板網站。
- 人像有親和與專業感，主視覺能建立信任。

手機狀態：PASS with minor notes。

優點：

- 手機首屏中人像完整，植物與愛心插圖位置可讀。
- 主標題卡片下接，整體情緒穩定。

注意：

- 手機版人像與插圖偏滿，若未來增加更多視覺元素，容易變得擁擠。
- 首頁小插圖偏可愛，應維持少量點綴，不宜再擴張成主要風格。

### 心理專欄列表

桌機狀態：PASS，作為目前文章視覺標準。

優點：

- 大標、插圖、作者卡組合完整。
- 專欄主視覺成熟、溫暖、安定。
- 留白與圖文比例佳，整體質感明顯高於一般個人網站。

手機狀態：PASS with follow-up。

優點：

- 文字層級清楚，CTA 清楚。
- 首屏閱讀感舒服。

注意：

- 手機首屏幾乎以文字為主，主視覺只在下緣露出一部分。
- 若希望手機版第一眼更有「質感」與記憶點，可評估讓文章主視覺稍微提前露出。

### 職場系列頁

桌機狀態：PASS。

優點：

- 標題非常有辨識度。
- 系列圖位置與閱讀提示卡搭配穩定。
- 圖片裁切沒有破壞主體，整體成熟。

手機狀態：PASS with follow-up。

優點：

- 標題、說明與 CTA 排版清楚。
- 情緒不壓迫，閱讀入口明確。

注意：

- 手機首屏的系列圖在下緣才出現，視覺吸引力延後。
- 若要增加手機首屏質感，可降低 hero 文字區高度或讓圖片更早進入畫面。

### 成癮單篇文章

桌機狀態：PASS as reading page。

優點：

- 乾淨、安靜，適合長文閱讀。
- 標題層級清楚，沒有混亂裝飾。

手機狀態：PASS as reading page, but weak brand memory。

優點：

- 文字可讀性好。
- 標籤與作者資訊清楚。

注意：

- 單篇文章首屏沒有文章主題圖，視覺記憶點弱。
- 對 SEO 分享與社群預覽而言，未來每篇文章應考慮主題圖或至少系列母圖變體。

### 講座合作

桌機狀態：PASS for homepage section, not enough for standalone landing page。

優點：

- 目前作為首頁段落清楚、乾淨。
- 合作流程卡片明確，CTA 可辨識。

手機狀態：PASS for current state。

注意：

- 目前缺少講座／活動專屬視覺。
- 若未來要獨立為可索引頁，質感需要再上升一級：加入專屬主視覺、活動情境圖、合作主題分區與實例／示意素材。

## 品牌一致性 QA

目前通過：

- 色彩系統一致：暖米色、棕色、鼠尾草綠與柔和粉橘能維持心理師品牌氣質。
- 字體層級一致：大標有記憶點，內文可讀。
- 插圖風格一致：專欄與系列圖成熟，適合作為後續主視覺標準。
- 人像可信任：專業照清楚、親近，能支撐首頁品牌。

目前風險：

- 首頁小插圖比專欄插圖更可愛，未來新增插圖時要以專欄系列圖的成熟度為主，不要往兒童化方向擴張。
- 單篇文章視覺較弱，未來文章越多，會讓每篇的品牌辨識度不足。
- 講座合作尚未形成獨立視覺系統。

## 修改建議

### P1：手機首屏視覺節奏微調

建議由版面體驗 AI 評估：

- 文章列表手機版是否讓主視覺更早露出。
- 系列頁手機版是否讓系列圖更早露出。

驗收標準：

- 不犧牲標題可讀性。
- 首屏底部至少看得到明確主視覺，而不是只露出一點邊角。

### P2：單篇文章主題圖策略

建議由 SEO 策略 AI 與插圖企劃 AI 討論：

- 短期：每個系列共用系列圖。
- 中期：每篇文章建立系列母圖變體。
- 長期：重要文章建立獨立主題圖。

驗收標準：

- 單篇文章首屏或分享預覽有更強辨識度。
- 不破壞目前溫暖、成熟、低壓迫的品牌氣質。

### P2：講座合作獨立視覺系統

建議由講座合作內容 AI 與網站品質總監 AI 討論：

- 是否建立獨立講座 landing page。
- 是否新增講座合作主視覺與活動示意圖。
- 是否加入實際活動照片或示意素材。

驗收標準：

- 看起來像專業合作邀約頁，而不是首頁段落展開版。
- 保留心理師品牌的安定感，不做過度商業化視覺。

## 本輪結論

目前網站質感狀態：可上線、可公開、具備專業個人品牌感。

不需要立即 hotfix。

需要優先處理的不是「修壞掉的畫面」，而是「讓手機首屏與單篇文章的視覺記憶點更強」。
